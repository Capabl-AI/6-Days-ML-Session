# 6 Days Machine Learning Session

## 📚 Overview

This repository contains a comprehensive 6-day machine learning curriculum designed to introduce learners to fundamental ML concepts, algorithms, and applications. The course progresses systematically from basic Python programming through advanced deep learning techniques, with hands-on Jupyter notebooks and real-world datasets.

## 🎯 Course Objectives

- Understand Python fundamentals and data manipulation
- Perform exploratory data analysis (EDA) on real datasets
- Implement supervised learning algorithms (Regression & Classification)
- Apply unsupervised learning techniques (Clustering)
- Master NLP concepts and text processing
- Build and train deep learning models

## 📅 Curriculum Flow

### **Day 1: Foundations & Regression**
**Topics:** Python basics, data exploration, supervised learning with regression
- **Day_1_Basic_Python_Session.ipynb** - Python fundamentals, libraries (NumPy, Pandas), data structures
- **Day_1_EDA.ipynb** - Exploratory Data Analysis techniques and visualization
- **Regression/** - Linear and advanced regression models
  - *Datasets:* `train.csv`, `test.csv`

### **Day 2: Classification**
**Topics:** Classification algorithms for binary/multi-class prediction
- **Classification.ipynb** - Decision trees, logistic regression, ensemble methods, model evaluation
- *Datasets:* `train.csv`, `test.csv`

### **Day 3: Clustering**
**Topics:** Unsupervised learning and pattern discovery
- **Clustering.ipynb** - K-Means, hierarchical clustering, customer segmentation
- *Dataset:* `Mall_Customers.csv` - Real mall customer data for segmentation analysis

### **Day 4: Natural Language Processing (NLP)**
**Topics:** Text processing, tokenization, feature extraction, and text classification
- **Natural_Language_Processing.ipynb** - Text preprocessing, TF-IDF, word embeddings, sentiment analysis
- *Datasets:* `train.csv`, `test.csv`

### **Day 5: Deep Learning**
**Topics:** Neural networks, deep learning architectures, and advanced techniques
- **Deep_Learning.ipynb** - Artificial Neural Networks, CNNs, RNNs, model optimization
- *Dataset:* `data.csv`

## 🛠️ Requirements

### Software & Libraries
- **Python** 3.7+
- **Jupyter Notebook** or **JupyterLab**
- **Core Libraries:**
  - `pandas` - Data manipulation and analysis
  - `numpy` - Numerical computations
  - `scikit-learn` - Machine learning algorithms
  - `matplotlib` & `seaborn` - Data visualization
  - `nltk` - Natural language processing
  - `tensorflow` / `keras` - Deep learning
  - `scipy` - Scientific computing

### Installation
```bash
pip install pandas numpy scikit-learn matplotlib seaborn nltk tensorflow keras scipy
```

## 🚀 Getting Started

1. **Clone/Download** this repository
2. **Install dependencies** (see Requirements section)
3. **Navigate** to the course directory

## 📝 How to Use This Repository

1. **Read notebooks sequentially** - Each day builds on previous concepts
2. **Execute cells** - Run code cells to understand implementation details
3. **Modify & experiment** - Change parameters, try different algorithms, explore data
4. **Complete exercises** - Work through problems at the end of each notebook
5. **Review datasets** - Understand data structure and preprocessing requirements

## 🔧 Best Practices

- **Run notebooks in order** to avoid missing foundational concepts
- **Understand, don't memorize** - Focus on why algorithms work, not just code
- **Experiment with hyperparameters** - See how different settings affect model performance
- **Document your findings** - Add comments and markdown cells explaining your observations
- **Work with real data** - Use provided datasets to practice on realistic ML problems
